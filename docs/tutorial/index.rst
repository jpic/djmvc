Tutorial
~~~~~~~~

This tutorial builds a djmvc application from the ground up. Each stage is a
Django app in ``djmvc_example``; source files are included literally and
validated by ``pytest -m tutorial``.

Optional packages (``djmvc_history``, ``djmvc_debug``, …) are covered in
:doc:`../install` — they work as soon as you add them to ``INSTALLED_APPS``.
